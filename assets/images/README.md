# Carpeta de Imágenes

## Estructura

```
images/
├── logo-ipn.png          # Logo oficial del IPN
├── logo-cocendi.png      # Logo de COCENDI
├── logo-equipo.png       # Logo del equipo (usar como favicon)
└── slider/               # Imágenes para el slider de noticias
    ├── noticia1.jpg
    ├── noticia2.jpg
    └── noticia3.jpg
```

## Instrucciones

1. **Logo del IPN**: Descargar logo oficial del Instituto Politécnico Nacional
2. **Logo COCENDI**: Obtener logo de la Coordinación de Centros de Desarrollo Infantil
3. **Logo del Equipo**: Crear logo personalizado del Equipo 4
4. **Imágenes del Slider**: Agregar mínimo 3 imágenes para noticias de COCENDI

## Formato Recomendado

- **Logos**: PNG con fondo transparente
- **Slider**: JPG o PNG, resolución recomendada 1200x400px
- **Favicon**: ICO o PNG 32x32px o 64x64px
